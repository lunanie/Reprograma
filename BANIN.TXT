Projeto 
Programa 1 - Cálculo de Médias Descrição do Projeto
Informações Iniciais
Este projeto estará baseado na leitura de um arquivo texto de entrada contendo dados de alunos de uma disciplina da faculdade. O conteúdo deste arquivo estara formatado como CSV (Comma Separated Values). Ele terá o nome ALUNOS.TXT e conterá dados conforme o layout mostrado abaixo. Nos campos numéricos reais lembre-se de usar o caractere ponto (.) como separador decimal.: 
181312202;Beatriz de Oliveira;7.3;8.8;5.9 
181312198;Carlos Alberto Mancini;4.4;3.7;7.6 
181212155;Eduardo Mendes Junqueira;7.9;5.3;6.1

Como pode ser visto acima, cada linha do arquivo refere-se a um aluno. Para os testes do programa deve-se usar um arquivo com pelo menos 25 alunos, ou seja, este arquivo terá pelo menos 25 linhas e as informações de cada linha estão descritas na tabela a seguir. É necessário que vocês criem esse arquivo usando um editor de texto simples corno o bloco de notas, por exemplo. 

Posição	|Informação			        |Formato		        | Observações 
(1)	    |N. de matricula do aluno	|9 dígitos numéricos	|
(2) 	|Nome do aluno			    |Texto			        |Este campo terá tarnanho variável, contendo quantos caracteres forem necessários para cada nome de aluno
(3) 	|Nota P1			        |Número Real		    |Nota da prirneira prova
(4) 	|Nota P2			        |Número Real		    |Nota da segunda
(5) 	|Nota MT			        |Número Real		    |Nota média dos trabalhos entregues

   
Pede-se neste Projeto Programa 
O resultado que se espera deste programa é a exibição em tela da situação de cada aluno no padrão mostrado abaixo 

Matrícula	|Nome do Aluno		    	|Média Final	|Situação

181312202	|Beatriz de Oliveira		|7.6		    |Aprovado 
181312198	|Carlos Alberto Mancini		|4.8 		    |Reprovado
181212155 	|Eduardo Mendes Junqueira 	|6.5		    |Aprovado 

O cálculo da média final é feito conforme a fórmula:

MF = (4.P1+4.P2+2.MT)/10 

E o critério para aprovação é que a média final seja maior ou igual A 6.0 












Projeto Programa 2— Totalização simples de vendas de produtos 

Descrição do Projeto 

Informações Iniciais 
Este projeto estará baseado na leitura de um arquivo texto de entrada contendo código, quantidade e valor unitário de produtos vendidos. O conteúdo deste arquivo estará formatado como CSV (Comma Separated Values). Ele terá o nome VENDAS.TXT e conterá dados conforme o layout mostrado abaixo. Nos campos numéricos reais lembre-se de usar o caractere ponto (.) corno separador decimal.: 

12455;14;17.90 
11590;130;4.50 
18300;12;16.43 
12455;96;17.43 
14570;50;6.12 
...


Como pode ser visto acima, cada linha do arquivo refere-se a um aluno. Para os testes do programa utilize o arquivo de dados fornecido pelo professor juntamente com este enunciado. As informações de cada linha do arquivo estão descritas na tabela abaixo: 

Posição |Informação         |Formato                |Observações  
(1)     | Código do produto |5 digitos numéricos    |Os códigos de produtos serão números entre 10000 e 21000 
(2)     |Quantidade         |Número inteiro  
(3)     |Preço unitário     |Número Real 

Pede-se neste Projeto Programa 
O programa deve ler os dados do arquivo de entrada, carregá-los em memória usando lista, calcular e mostrar na tela o total geral vendido. Em seguida deve permanecer em laço enquanto não for digitado zero (0). Para cada repetição do laço deve-se ler um código de produto, obrigatoriamente no intervalo (10000 e 21000) e mostrar na tela o total vendido para aquele produto.

>> Digite o código: 14330 
Total vendido do produto 14330 = R$ 7326.14 
>> Digite c código: 11200 
Total vendido do produto 11200 - R$ 0.00 
>> Digite o código: 55000 
55000 Código inválido 
>> Digite o código: 14500 
Total vendido do produto 14500 = R$ 712.40 
>> Digite o código: O 
Fim do programa
